# Fynodoro

Fynodoro is the little widget for doing Pomodoro.

## Development

Run:

```shell
go generate && go run .
```

Package (Requires: `go get fyne.io/fyne/v2/cmd/fyne`):

```shell
fyne package -os linux -icon assets/Icon.png
```
